D.S.Reader for iOS

Get from AppStore

E-Book Reader supporting EPUB, PDF, and CBZ formats, with custom fonts and custom dictionary support.

When paired with calibre Content Server, you can easily browse your libraries, download books for reading.

It will also track and sync you reading progress and highlights across all devices paired with the same server.

Installation Notes

After installing and restarting calibre, please open plugin option to let it finish setting its custom columns.

By default DSReader Helper will listen on port $(1 + calibre's port number). You can change it to another port number if needed, and please make sure it is open for access if you have firewall.

Functionalities

calibre has a good wealth of great plugins, and we want to make use of them to enrich DSReader's reading experiences. However, plugins' functionalities are not exposed by calibre context server APIs. Therefore we have made a helper plugin for this purpose.

Through this plugin DSReader will be able to:

  • Pull other plugins' custom columns settings.
    • Currently supports "Count Pages" and "Goodreads Sync"
  • Tap into other plugins' functionalities.
    • Currently supports automatically updating reading progress and book shelf to Goodreads using "Goodreads Sync"
  • Lookup definitions from user supplied dictionaries.
  • Manage reading positions across multiple devices more effectively and robust.
DSReader Helper has reused calibre Content Server's HTTP service components. Therefore this port has the same safety measures as calibre itself. Restrictions to User account should work as intended.
