--- title: Instructions on Using Dictionary Library ---

Supported Dictionary Formats

Note: We are not affliliated with dictionary developer in any way.

Right now this plugin supports MDict format (with file extension .mdx & .mdd).

You can find many free dictionaries online, or build your own using MDXBuilder.

Tutorial

  • Create a dedicated calibre library (ie. "Dictionary").
  • Pack dictionaries into ZIP files, one file per dictionary. Add them to the library, and make meaningful titles.
  • Go to plugin preference's Dictionary tab, pick your library, click "Refresh" to prepare the dictionary list.
    • You can also tweak dictionary order by using corresponding buttons.
  • Finally go to App's "Settings" -> "Server & Library" -> "Connect & Update Config" to make App aware of that.
